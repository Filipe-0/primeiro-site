Como adicionar mais jogos

Para colocar jogos prontos (ex.: HTML5/JS), crie uma pasta no seu repositório (ex.: games/) e adicione os arquivos do jogo, depois adicione um link ou iframe no index apontando para games/nome-do-jogo/index.html.
Exemplo de link para jogo externo:
<iframe src="https://example.com/jogo.html"></iframe>
Dicas:
Se for usar imagens com espaços no nome, use %20 (ex.: imagen%20do%20meu%20site.jfif) ou renomeie sem espaços.
